# manovaspace

MIT TypeScript libraries extracted from generic tooling — reusable without the proprietary Orbit platform.

## Packages

| npm | Description |
| --- | --- |
| [`@manovaspace/tsconfig`](/packages/tsconfig/) | Strict shared TypeScript presets for Next.js and React libraries |
| [`@manovaspace/markdown`](/packages/markdown/) | Lightweight GFM markdown renderer for React and Next.js |
| [`@manovaspace/pwa`](/packages/pwa/) | Serwist and web app manifest helpers for Next.js PWAs |
| [`@manovaspace/observability`](/packages/observability/) | Next.js instrumentation helpers for Sentry |

## Design system

Tokens, UI, and devtools live in a separate repo: [manovaspace.github.io/design-system](https://manovaspace.github.io/design-system/).

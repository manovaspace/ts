`@manovaspace/ui` is the MIT-licensed design system layer: Radix primitives, Tailwind 4 styles, motion helpers, and icon packs.

```bash
npm install @manovaspace/ui @manovaspace/tokens
```

```tsx
import { Button } from "@manovaspace/ui";
import "@manovaspace/ui/styles.css";
```

Peer dependencies: `react` and `react-dom` ^19.2.

See the [source on GitHub](https://github.com/manovaspace/ts/tree/main/packages/ui).

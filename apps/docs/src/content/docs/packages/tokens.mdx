title: tokens
description: Design tokens — default.json, semantic CSS, and font assets.
---

`@manovaspace/tokens` provides the default token JSON, generated CSS variables, and font assets used by `@manovaspace/ui`.

```bash
npm install @manovaspace/tokens
```

```css
@import "@manovaspace/tokens/tokens.css";
```

See the [source on GitHub](https://github.com/manovaspace/ts/tree/main/packages/tokens).

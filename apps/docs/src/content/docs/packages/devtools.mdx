---
title: devtools
description: Development-only design system toolbar for Next.js apps.
---

`@manovaspace/devtools` adds a dev-only toolbar for editing design tokens and previewing theme changes. **Do not import in production bundles.**

```bash
npm install -D @manovaspace/devtools
```

```tsx
import { OrbitDevtools } from "@manovaspace/devtools";
```

Requires `@manovaspace/tokens` and `@manovaspace/ui`.

See the [source on GitHub](https://github.com/manovaspace/ts/tree/main/packages/devtools).
